# Exemplos de Execução — PoC OSINT

Esta pasta contém ficheiros de exemplo que demonstram a utilização da prova de conceito (PoC)
de enumeração de subdomínios incluída neste projeto.

## Conteúdo
- `subdomains_input_example.txt`  
  Lista de subdomínios (prefixos) utilizada como entrada para o script.

- `subdomains_output_example.txt`  
- `subdomains_active.txt`  
  Resultado da execução do script, contendo apenas os subdomínios que apresentaram
  resolução DNS válida.

## Nota Importante
- Os domínios e subdomínios apresentados são fictícios e utilizados apenas para fins
  académicos e demonstrativos.
- Nenhuma enumeração foi realizada contra sistemas reais sem autorização.
