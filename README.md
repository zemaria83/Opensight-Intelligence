# OpenSight Intelligence — Projeto OSINT

Plataforma conceptual e protótipo (Uizard) para prestação de serviços OSINT (Open Source Intelligence),
incluindo recolha, correlação e reporting de informação pública de forma ética e conforme RGPD.

## Objetivos
- Diagnóstico de exposição digital
- Relatórios OSINT e recomendações
- Protótipo de dashboard e fluxos
- PoC de automação: enumeração de subdomínios via DNS

## Conteúdos do repositório
- `docs/` — capítulos do relatório e anexos
- `design/` — wireframes e protótipo Uizard (assets e descrição)
- `diagrams/` — UML, fluxogramas e modelo de dados
- `scripts/` — PoCs (ex.: enumeração de subdomínios)
- `assets/` — imagens, screenshots e logótipos
- `examples/` — outputs de exemplo (sem dados sensíveis)

## Ética e Limitações
- Apenas fontes públicas e utilização autorizada.
- Sem hacking, exploração ou intrusão.
- Dados de exemplo são fictícios ou anonimizados.
